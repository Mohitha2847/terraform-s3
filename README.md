# Terraform S3 Bucket Project

## Overview

This project demonstrates Infrastructure as Code (IaC) using Terraform.

The project provisions an AWS S3 bucket using Terraform configuration files.

## Files

- main.tf
- variables.tf
- outputs.tf
- .gitignore

## Commands Used

### Check Terraform

terraform --version

### Initialize Terraform

terraform init

### Validate Configuration

terraform validate

### Generate Execution Plan

terraform plan

### Apply Configuration

terraform apply

### Destroy Resources

terraform destroy

## Note

The Terraform configuration was developed and validated locally.

AWS resource provisioning was not performed because AWS account access was unavailable.
